# Screen recording angular application

## Install
`git clone <url_repo>`

`cd <url_repo> && yarn`

### Run development live
`yarn start`

### Build production
`yarn build`

### Test
`yarn test`

## Docker

### Build
`docker build -t djm/ang-app:latest .`

### Run container
`docker run -d -p 80:80 djm/ang-app:latest`
